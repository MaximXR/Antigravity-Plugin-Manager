# Antigravity Plugin Manager

Визуальный менеджер плагинов и анализатор активного контекста для **Antigravity IDE**.

Расширение позволяет включать и выключать глобальные плагины одним кликом, а также отслеживать общее количество подключенных к ИИ-агенту навыков (Skills), правил (Rules) и воркфлоу (Workflows), предотвращая переполнение контекстного окна.

---

## Основные возможности

### 1. Управление глобальными плагинами (Вкл/Выкл)
* **Физическое перемещение**: Так как встроенный сканер Antigravity IDE игнорирует символические ссылки и Junctions (из-за внутренних особенностей проверки папок), отключение плагина выполняется путем **физического перемещения его папки** из активного каталога `plugins/` в специальное хранилище (Storage).
* **Мгновенная скорость**: По умолчанию хранилище расположено в той же родительской папке (`config/plugins_storage/`), поэтому перемещение выполняется на уровне файловой системы за доли миллисекунды (как обычное переименование записи в MFT) и не занимает дополнительного дискового пространства.
* **Custom Storage**: Папку хранилища можно изменить вручную прямо из интерфейса менеджера с помощью удобного диалогового окна.

### 2. Анализ активного контекста (Context Stats)
Панель статистики в реальном времени считывает и выводит:
* 🧩 **Plugins**: Количество активных/зарегистрированных плагинов.
* 🧠 **Skills**: Количество активных навыков (включая встроенные в плагины и глобальные/локальные папки `skills/`).
* 📜 **Rules**: Локальные правила текущего проекта (`.agents/rules/`).
* 📂 **Workflows**: Активные цепочки инструкций (глобальные воркфлоу + локальные воркфлоу проекта + встроенные в плагины).

### 3. Интеграция со Status Bar
* В нижнем меню выводится кнопка: `🧩 Плагины: X/Y`.
* При наведении на нее мыши отображается подробный Markdown-список со статусом каждого плагина (список активных и список выключенных).

---

## Настройки (Settings)

Параметры в `settings.json`:

* `antigravity-plugin-manager.storagePath`: Абсолютный путь к папке хранения отключенных плагинов (по умолчанию `~/.gemini/config/plugins_storage`).
* `antigravity-plugin-manager.language`: Язык интерфейса (`auto` — системный, `en` — английский, `ru` — русский).
